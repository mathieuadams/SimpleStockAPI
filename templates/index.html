📈 Stock Stats API

This API returns key technical indicators and stats for a given stock symbol using Yahoo Finance data.

📡 Endpoint

GET /api/v1/stats?symbol=XYZ

🔍 Parameters

  • symbol: Stock ticker symbol (e.g., AAPL, TSLA)

🧪 Example

/api/v1/stats?symbol=AAPL

📦 Example Response

{
  "symbol": "AAPL",
  "price": 172.45,
  "ma_10": 170.92,
  "ma_20": 169.84,
  "ma_50": 165.33,
  "correlation_to_SPY": 0.91,
  "rsi": 58.23,
  "macd": {
    "macd": 1.45,
    "signal": 1.21,
    "histogram": 0.24
  }
}

Built with Flask · Data from Yahoo Finance
